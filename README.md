# 智能小车控制系统

基于 Arduino/STM32 的智能小车控制系统，包含多种模块支持寻迹、避障等功能。该项目实现了不同场景下的智能控制算法。car13以及car14为小地图使用的模块，carchuangyi为复杂地图使用。

## 项目讲解
[blog.xcstudio.top](https://blog.xcstudio.top/index.php/2025/11/28/kaiyuan_ino/)

## 快速开始

根据不同硬件配置选择对应的模块：

### ESP8266 超声波模块
适用于基础避障功能的小车配置，使用红外传感器进行距离检测。
```bash
# 打开 esp8266/esp8266.ino 文件
# 使用 Arduino IDE 编译并上传到 ESP8266 开发板
```

### CAR13 标准 STM32 小车模块
标准配置，支持基本的寻迹功能，按照预设路径在10个节点间移动。
```bash
# 打开 car13/car13/car13.ino 文件
# 使用 Arduino IDE 编译并上传到 STM32 开发板
```

### CAR14 智能模块（car13拓展）
增强版智能控制模块，具有自适应路径判断能力，但似乎逻辑处理有部分问题仍在开发中。
```bash
# 打开 car14/car14.ino 文件
# 使用 Arduino IDE 编译并上传到 STM32 开发板
```

### CARCHUANGYI 大地图模块
专为复杂路径设计的大地图寻迹系统，包含寻黑线循白逻辑，支持11个节点的复杂路径导航。
```bash
# 打开 carchuangyi/carchuangyi.ino 文件
# 使用 Arduino IDE 编译并上传到 STM32 开发板
```

## 功能特性

- 多种硬件模块支持（ESP8266、STM32等）
- 智能寻迹功能（黑白线识别）
- 自适应路径规划与导航
- 节点检测与处理机制
- 多种转向策略（原地转、调整转、掉头等）
- 传感器数据融合处理
- 可扩展的模块化设计

## 模块说明

### ESP8266 超声波模块
- 基于 ESP8266 微控制器平台
- 集成超声波传感器用于距离检测
- 支持红外传感器检测
- LED状态指示功能

### CAR13 标准 STM32 小车模块
- 基于 STM32 微控制器的标准配置
- 支持红外传感器寻迹功能（4个传感器）
- 集成电机驱动控制（L298N驱动模块）
- 预设10个节点的路径导航
- 支持多种转向模式（直行、微调、原地转、掉头等）
- 串口调试信息输出

### CAR14 智能模块
- 增强型 STM32 控制模块
- 支持更复杂的算法处理
- 智能路径判断能力（基于侧边传感器）
- 分组节点处理机制（每3个节点为一组）
- 自适应转向完成检测
- 部分高级功能仍在开发中

### CARCHUANGYI 大地图模块
- 专为复杂路径设计（支持11个节点）
- 实现寻黑线循白逻辑
- 支持大范围路径规划与导航
- 包含智能决策算法
- 光线适应性处理
- 多种运动控制策略

## 安装说明

### 硬件要求
- STM32 或 ESP8266 开发板
- L298N 电机驱动模块
- 红外循迹传感器（多个）
- 超声波传感器（仅 ESP8266 模块需要）
- 直流电机（2个）
- 小车底盘

### 软件要求
- Arduino IDE 1.8.0 或更高版本
- 对应开发板的支持包（STM32 或 ESP8266）

### 安装步骤
```bash
# 1. 克隆或下载项目代码

# 2. 打开对应模块的 .ino 文件
# 例如：打开 car13/car13/car13.ino

# 3. 在 Arduino IDE 中选择正确的开发板和端口

# 4. 编译并上传代码到开发板
```

## 使用方法

### 基本使用
1. 按照电路图连接所有硬件组件
2. 将小车放置在循迹地图上
3. 打开串口监视器（波特率设置为115200）
4. 给小车上电，等待3秒倒计时后开始运行

### 调试信息
通过串口监视器可以查看：
- 传感器实时状态
- 当前执行的动作
- 节点检测信息
